## Picker Editor

**ピッカーのボタンを編集するエディタです。**

## エディタ

**ボタンモジュールの位置や大きさ、角度をインタラクティブに調整します。**

- 左ボタンクリック : 選択
- 左ボタンドラッグ : 移動
- 右ボタンドラッグ : 回転
- ホイールドラッグ or スクロール : スケール

---

## モジュールリスト

**モジュールを選択、表示、ロックするリストです。**

- 👁️✓ : モジュールを表示します。
- 🔒️✓ : モジュールをロックします。
- 右クリック : 右クリックメニューを表示します。

---

## Transform Controls

**ボタンモジュールの位置や大きさ、角度を数値でに調整します。**

- Translate X : 横軸の位置
- Translate Y : 縦軸の位置
- Rotate : 回転
- Scale : スケール
- Local Flip : ローカル座標軸での反転
- World Flip : ワールド座標軸での反転

---

## Enter Child Level

**子モジュールを編集するエディタを開きます。**
モジュールに + 状のボタンがある場合、そのモジュールには子モジュールが存在している印です。

---

## Reload Data

**モジュールの追加や削除等で、エディタとシーンの情報に差異が生まれた際に、エディタをシーンに同期します。**
